# Teslo API

1. Tải dự án
2. `yarn install`
3. Sao chép tên file `.env` và đổi tên thành `.env.tempalte`
4. Thay đổi các biến môi trường
5. Kết nối cơ sở dữ liệu

```
docker-compose up -d
```

6. thuc seed

```
http://localhost:3000/api/seed
```

7. Chạy dự án: `yarn start:dev`
